# Portfolio / CV – Clément Rouchit

Site statique (HTML/CSS/JS) conçu pour GitHub Pages.

## Structure
- `index.html` : page d'accueil
- `competences.html` : compétences + liens vers projets
- `projets.html` : projets + liens vers compétences
- `cv.html` : CV PDF + aperçu
- `contact.html` : contact

## Mise en ligne sur GitHub Pages
1. Créez un dépôt GitHub, puis ajoutez ces fichiers.
2. Dans GitHub : **Settings → Pages**.
3. Source : `Deploy from a branch` → `main` → `/ (root)`.

## À personnaliser
- Ajoutez votre photo : `assets/img/photo.jpg`.
- Vérifiez le nom du CV dans `assets/cv/` (actuellement : `CV_Clément_Rouchit_Alternance_2025.pdf`).
- Mettez vos liens LinkedIn/GitHub dans `index.html`.
- Ajoutez des preuves : captures, liens de dépôts, schémas.
